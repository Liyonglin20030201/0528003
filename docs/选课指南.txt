华智大学本科生选课指南（2024-2025学年）

一、选课时间安排

1. 预选阶段：每学期第16周周一至周五（上学期末进行下学期选课）
   - 预选不受人数限制，所有学生均可选择心仪课程。
   - 预选结束后，系统将根据选课优先级进行抽签。

2. 正选阶段：每学期第17周周一至周三
   - 预选未中签的学生可在正选阶段补选其他课程。
   - 正选阶段按先到先得原则，额满为止。

3. 补退选阶段：每学期开学第1周周一至第2周周五
   - 学生可以退选已选课程或补选有余量的课程。
   - 补退选阶段结束后，选课结果不可更改。

4. 期中退课：第8周周一至周五
   - 学生因特殊原因可申请退课，退课记录将在成绩单上标注"W"。
   - 每学期最多退课1门，累计不超过3门。

二、选课规则

1. 学分要求
   - 每学期选课学分不得低于15学分（毕业年级最后一学期除外）。
   - 每学期选课学分不得超过28学分。
   - 如需超学分选课，须提前向学院教务办申请，经批准后方可执行。

2. 先修课程要求
   - 部分课程设有先修课程要求，未修读先修课程的学生无法选择该课程。
   - 先修课程要求可在教务系统课程详情页面查看。

3. 课程冲突处理
   - 上课时间冲突的课程不可同时选择。
   - 考试时间冲突的课程原则上不可同时选择，如确有需要，须在选课前向教务处申请考试时间调整。

4. 重修选课
   - 必修课不及格须重修，重修课程优先选择。
   - 重修课程与正常课程时间冲突时，可申请免听，但须按时参加考试。
   - 重修选课在补退选阶段第2周进行。

三、选课优先级规则

1. 专业必修课 > 专业选修课 > 通识选修课
2. 高年级学生 > 低年级学生（同一课程）
3. 毕业年级学生享有最高优先级

四、跨专业/跨院选课

1. 学生可以跨专业、跨学院选课，但须满足该课程的先修要求。
2. 跨专业选课学分计入通识选修学分。
3. 热门跨专业课程可能设有专业学生优先的名额限制。

五、体育课选课

1. 大一、大二学生每学期必须选修一门体育课。
2. 同一体育项目不可重复选修。
3. 因身体原因无法参加正常体育课的学生，可凭三甲医院证明申请保健体育课。

六、选课系统操作说明

1. 登录地址：http://jwxt.huazhi.edu.cn
2. 登录账号：学号
3. 初始密码：身份证后六位
4. 选课操作路径：教务系统 → 选课中心 → 本学期选课

七、常见问题

Q：选课系统打不开怎么办？
A：选课高峰期系统可能拥堵，建议错峰选课。也可尝试更换浏览器（推荐Chrome或Firefox）。

Q：选课后可以换老师吗？
A：补退选阶段可以退选后重新选择其他教学班。正选结束后不支持换班。

Q：学分不够怎么办？
A：可在补退选阶段补选课程。如确实无法达到最低学分要求，需向学院提交情况说明。

八、注意事项

1. 选课前请仔细阅读培养方案，了解各模块学分要求。
2. 请关注教务处通知，了解选课时间和相关政策变化。
3. 选课结果以教务系统最终显示为准。
4. 如有选课相关问题，请联系所在学院教务办或学校教务处（电话：010-12345678）。
